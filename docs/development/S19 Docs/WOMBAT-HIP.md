## Executive Summary

Hip roofs are **polyhedral forms**, not prismatic extrusions. Unlike gable/shed roofs which can be generated by extruding a 2D profile, hip roofs require a **clipping plane approach** where we:

1. **Generate the gable solution** (already working)
2. **Clip the gable ends** with planes from corner eaves to ridge endpoints
3. **Recalculate the resulting volume** (smaller than gable due to clipped pyramidal caps)
4. **Maintain the same roof pitch** (x:12 ratio preserved from gable)

**Key Insight from Backup**: The hip roof ridge bisects the corners at **spread s=0.5** (equivalent to 45° in classical terms), which determines the ridge endpoint positions using pure rational trigonometry.

---

## Mathematical Foundation (Rational Trigonometry)

### Spread vs. Angle Convention

**Critical**: We use **spread (s)** instead of angles:
- Spread s = sin²(θ)
- For 45°: s = sin²(45°) = 0.5
- This is the **bisection spread** for hip roof corners

### Hip Roof Geometry Constraints

Given:
- Footprint: width (W) × length (L), where L ≥ W
- Roof area constraint: A_roof (from d_85)
- Gable pitch already solved: rise:12 ratio

Derive:
1. **Ridge length** (R): Determined by 45° hip rafters (s=0.5 bisection)
2. **Ridge height** (h): Same as gable height (preserves pitch)
3. **Hip rafter positions**: From corner eaves to ridge endpoints

## Gable-Then-Clip Strategy

### Phase 1: Solve Gable Baseline

Using existing `solveGable2DProfile()` logic:

```javascript
// Already implemented in Section19.js
const gableData = calculateGableHeight(width, length, roofArea);

// Extract gable parameters
const gableHeight = gableData.height;
const gableRidgeLength = Math.max(width, length);  // Full ridge (long dimension)
const span = Math.min(width, length);              // Short dimension
```

**Result**: Full gable roof with ridge running entire length of building.

### Phase 2: Determine Hip Clipping Planes

The hip roof **truncates** the gable by clipping triangular pyramidal caps from each end.

**Clipping Plane Position** (Rational Trigonometry):

For a **bisection at spread s=0.5** (45° hip rafters):
- Hip rafter travels equal horizontal distances in X and Y directions
- From corner eave (±W/2, ±L/2, z=h_eave)
- To ridge endpoint (0, ±L_ridge_end, z=h_ridge)

**Ridge Shortening Calculation**:

```javascript
// Hip roof ridge is SHORTER than gable ridge
// Clipping occurs at distance (span/2) from each end along long axis

const span = Math.min(width, length);           // Short dimension
const maxDimension = Math.max(width, length);   // Long dimension

// Ridge endpoints move inward by (span/2) from each end
const ridgeShortening = span;  // Total shortening (span/2 at each end)
const hipRidgeLength = maxDimension - ridgeShortening;

// For SQUARE buildings: hipRidgeLength = 0 (pure pyramid)
```

**Geometry**:
```
Plan View (rectangular building):
┌─────────────────────────────┐
│╲                          ╱ │ ← Hip rafters (s=0.5)
│ ╲                        ╱  │
│  ╲                      ╱   │
├   ──────────────────────    ┤ ← Ridge (shortened)
│  ╱                       ╲  │
│ ╱                         ╲ │
│╱                           ╲│
└─────────────────────────────┘

Ridge start: (0, -hipRidgeLength/2, h_ridge)
Ridge end:   (0, +hipRidgeLength/2, h_ridge)
```

### Phase 3: Calculate Hip Roof Volume

Hip roof volume = Gable volume - 2 × Pyramidal cap volume

**Pyramidal Cap** (at each end):
- Base: Triangle with corners at eave level
  - Vertices: (±W/2, L_end, h_eave) and (0, L_ridge_end, h_eave)
  - Base area: (W × (L_end - L_ridge_end)) / 2 = (W × span/2) / 2 = W×span/4
- Apex: Ridge endpoint (0, L_ridge_end, h_ridge)
- Height: h_ridge - h_eave

```javascript
// Pyramidal cap volume (one end)
const baseTriangleArea = (span / 2) * (span / 2);  // Right triangle at end
const pyramidHeight = gableHeight;                  // Same as gable peak height
const capVolume = (1/3) * baseTriangleArea * pyramidHeight;

// Total hip roof volume
const gableVolume = (footprintArea * gableHeight) / 2;
const hipVolume = gableVolume - 2 * capVolume;
```

**Simplification**:
```javascript
// For rectangular base (W × L):
const gableVolume = (W * L * h) / 2;
const capVolume = (1/3) * (span²/4) * h;  // Each cap
const hipVolume = (W * L * h) / 2 - (2/3) * (span²/4) * h;

// Factor out h:
const hipVolume = h * ((W * L) / 2 - (span²) / 6);
```

## Roof Area Constraint Satisfaction (Rational Trigonometry)

**Challenge**: Solve for height quadrance Q such that hip roof area = d_85 (target roof area).

**Philosophy** (per [PRISMATIC-TERMINOLOGY.md](./PRISMATIC-TERMINOLOGY.md)):
- Work in **whole numbers** (ratios/quadrances) as long as possible
- Defer square roots until final step
- Use **spread** (s) instead of angles
- Preserve numerical precision through rational operations

---

### Case 1: Square Building (Pure Pyramid) - DIRECT SOLUTION ✅

For square footprint (W = L = a):

**Area Formula**:
```
A_pyramid = 4 × (triangle area)
          = 4 × (a × slopeLength / 2)
          = 2a × slopeLength
```

**Direct Algebraic Solution** (no iteration needed):

```javascript
// Given: footprint side length a, target area A_target
const a = width;  // Square: width = length
const A_target = roofArea;  // From d_85

// Step 1: Solve for slope length (rational division)
const slopeLength = A_target / (2 * a);

// Step 2: Work in quadrance space (stay rational!)
const Q_slope = slopeLength * slopeLength;      // R in Wildberger notation
const Q_halfBase = (a / 2) * (a / 2);           // Quadrance of half-base
const Q_height = Q_slope - Q_halfBase;          // Pythagorean theorem (Q-space)

// Step 3: Validate (check spread)
const spread = Q_halfBase / Q_slope;            // s = Q_base / R
if (spread >= 1.0) {
  // Impossible geometry: roof area too small for pyramid
  console.error(`[WOMBAT] Pyramid impossible: spread s = ${spread.toFixed(3)} ≥ 1.0`);
  return { roofType: "flat", roofHeight: 0 };
}

// Step 4: Extract height (single sqrt at end)
const roofHeight = Math.sqrt(Q_height);         // Only irrational operation

console.log(`[WOMBAT] Pyramid (square) solved directly:`);
console.log(`  Side: ${a.toFixed(2)}m`);
console.log(`  Slope length: ${slopeLength.toFixed(2)}m`);
console.log(`  Spread s: ${spread.toFixed(3)}`);
console.log(`  Height Q: ${Q_height.toFixed(2)} m²`);
console.log(`  Height: ${roofHeight.toFixed(2)}m`);
```

**Result**: Pure pyramid height solved in **constant time** using pure rational trigonometry! ✨

### Case 2: Rectangular Building (Hip Roof) - THREE APPROACHES

For rectangular footprint (L > W):

**Ridge geometry** (deterministic, no solving needed):
```javascript
const ridgeLength = L - W;   // Ridge shortened by W (span/2 at each end)
const span = W;               // Short dimension
```

**Area Formula** (function of height quadrance Q):
```
A_hip(Q) = 2·ridgeLength·√(Q + W²/4) + 2W·√(Q + W²/2)
         = 2(L-W)·√(Q + W²/4) + 2W·√(Q + W²/2)
```

This is a **single-variable equation in Q**, but contains nested square roots.

**The Challenge**: Solve `A_hip(Q) = A_target` for Q.

---

#### Option A: Newton-Raphson (Fast Numerical, ~2 iterations)

Use Newton's method: `Q_new = Q_old - f(Q)/f'(Q)`

Where:
```
f(Q) = A_hip(Q) - A_target
f'(Q) = derivative of A_hip with respect to Q
```

**Pros**:
- Very fast (1-2 iterations typical)
- Quadratic convergence
- Still numerically stable

**Cons**:
- Requires derivative calculation
- More complex code

**Implementation**:
```javascript
// Derivative of A_hip(Q):
// d/dQ [2(L-W)·√(Q + W²/4)] = (L-W) / √(Q + W²/4)
// d/dQ [2W·√(Q + W²/2)] = W / √(Q + W²/2)

function f(Q) {
  const mainSlope = 2 * ridgeLength * Math.sqrt(Q + W*W/4);
  const hipEnds = 2 * W * Math.sqrt(Q + W*W/2);
  return mainSlope + hipEnds - A_target;
}

function f_prime(Q) {
  const term1 = ridgeLength / Math.sqrt(Q + W*W/4);
  const term2 = W / Math.sqrt(Q + W*W/2);
  return term1 + term2;
}

// Newton-Raphson iteration
let Q = initialGuess;  // Start with gable Q
for (let i = 0; i < 5; i++) {  // Rarely needs more than 2-3
  Q = Q - f(Q) / f_prime(Q);
  if (Math.abs(f(Q)) < 0.01) break;  // Converged
}

const roofHeight = Math.sqrt(Q);
```

#### Option B: Binary Search (Simple, Proven)

This is what the backup uses. Simple bisection on Q until converged.

**Pros**:
- Dead simple to implement
- Guaranteed convergence
- No derivatives needed
- Proven in production (backup code)

**Cons**:
- Slower than Newton-Raphson (~10-20 iterations vs 2-3)
- Linear convergence (not quadratic)

**Implementation**:
```javascript
let Q_min = 0;
let Q_max = (span * 2) * (span * 2);  // Upper bound
const tolerance = 0.01;  // 1cm² area precision

for (let i = 0; i < 50; i++) {
  const Q = (Q_min + Q_max) / 2;

  // Calculate area with this Q
  const mainSlope = 2 * ridgeLength * Math.sqrt(Q + W*W/4);
  const hipEnds = 2 * W * Math.sqrt(Q + W*W/2);
  const area = mainSlope + hipEnds;

  if (Math.abs(area - A_target) < tolerance) break;

  if (area < A_target) {
    Q_min = Q;  // Need larger Q (taller roof)
  } else {
    Q_max = Q;  // Need smaller Q (flatter roof)
  }
}

const roofHeight = Math.sqrt(Q);
```

#### Option C: WINNER!! Clever Algebraic Substitution (RT Purist) 🎯

**Goal**: Find a rational substitution that eliminates nested square roots, using geometric reasoning like Pythagoras, Buckminster Fuller, or Wildberger.

**Key Observation from Backup** ([Section19.js.backup:877-1024](../../src/sections/Section19.js.backup)):

The backup uses **binary search on height** (10-20 iterations), not a direct solution:
```javascript
// Backup approach (lines 949-990):
while (hMax - hMin > tolerance && iteration < maxIterations) {
  const h = (hMin + hMax) / 2;

  // Calculate area with this height
  const slopeLength = Math.sqrt(h*h + (span/2)*(span/2));
  const mainSlopeArea = 2 * ridgeLength * slopeLength;

  const hipRafterQuadrance = (span/2)*(span/2) + ridgeOffset*ridgeOffset + h*h;
  const hipRafterLength = Math.sqrt(hipRafterQuadrance);
  const hipEndArea = 2 * span * hipRafterLength;

  const calculatedArea = mainSlopeArea + hipEndArea;

  // Adjust search bounds
  if (calculatedArea < targetRoofArea) hMin = h;
  else hMax = h;
}
```

This is **numerical iteration**, not geometric insight. Can we do better?

---

**Geometric Insight: Hip = Gable - 2×Tetrahedra**

The user's key insight:
> "We have a solver for the gable roof. We clip what are essentially two tetrahedra from each end of the gable to derive the hip roof. If we know what the volume of the tetrahedron is, we subtract 2×tetVol from Gable Volume. But what we are after is Q, to solve for the area constraint."

**The Challenge**:
- Gable solver works from **area constraint** → height
- Tetrahedral clip changes **volume**, not area directly
- We need: **area constraint** → height (for hip)

**Can we express hip area in terms of gable area?**

Let's analyze:

```
Gable area (full length):
A_gable = 2·L·√(Q + W²/4)

Hip area (shortened ridge + end caps):
A_hip = 2·(L-W)·√(Q + W²/4) + 2W·√(Q + W²/2)
      = A_gable - 2W·√(Q + W²/4) + 2W·√(Q + W²/2)
      = A_gable + 2W·[√(Q + W²/2) - √(Q + W²/4)]
```

**Key relationship**:
```
√(Q + W²/2) = √[(Q + W²/4) + W²/4]
            = √[u² + W²/4]    where u = √(Q + W²/4)
```

This is a **Pythagorean relationship** in disguise! The hip rafter length is the hypotenuse of a right triangle with legs:
- u = main slope slant height
- W/2 = half the ridge offset

**Substitution Strategy**:

Let `u = √(Q + W²/4)` (main slope slant height from gable solver)

Then:
```
A_hip = 2(L-W)·u + 2W·√(u² + W²/4)
```

**Single-variable equation in u**, but still has nested sqrt.

To eliminate the nested radical, square strategically:
```
A_hip - 2(L-W)·u = 2W·√(u² + W²/4)

[A_hip - 2(L-W)·u]² = 4W²·(u² + W²/4)
[A_hip - 2(L-W)·u]² = 4W²·u² + W⁴

A_hip² - 4(L-W)·A_hip·u + 4(L-W)²·u² = 4W²·u² + W⁴

A_hip² - 4(L-W)·A_hip·u + 4(L-W)²·u² - 4W²·u² = W⁴

A_hip² - 4(L-W)·A_hip·u + [4(L-W)² - 4W²]·u² = W⁴
```

Factor out 4:
```
4(L-W)² - 4W² = 4[(L-W)² - W²]
                = 4[L² - 2LW + W² - W²]
                = 4[L² - 2LW]
                = 4L(L - 2W)
```

**Final quadratic in u**:
```
4L(L-2W)·u² - 4(L-W)·A_hip·u + (A_hip² - W⁴) = 0
```

**Quadratic formula solution**:
```
u = [4(L-W)·A_hip ± √[16(L-W)²·A_hip² - 16L(L-2W)·(A_hip² - W⁴)]] / [8L(L-2W)]

u = [(L-W)·A_hip ± √[(L-W)²·A_hip² - L(L-2W)·(A_hip² - W⁴)]] / [2L(L-2W)]
```

**Then extract Q**:
```
Q = u² - W²/4
```

**Verdict**: We **CAN** solve it algebraically via quadratic formula! This is more "geometric" than Newton-Raphson (no derivatives), but still involves a square root in the discriminant.

**Is this simpler than Newton-Raphson?**
- Quadratic formula: 1 evaluation (but complex discriminant)
- Newton-Raphson: 2-3 iterations (but simpler per-iteration)

**Trade-offs**:
- Algebraic (quadratic): True one-shot solution, more RT-flavored
- Newton-Raphson: Iterative but conceptually cleaner code
- Binary Search: Proven, simple, but slowest

**Wildberger's approach**: Would likely prefer the quadratic solution (stays in Q-space longer, single sqrt at end).

### Recommendation: Pure Algebraic Approach ✨

**For Square Buildings** (W = L):
- Use **direct algebraic solution** (Case 1 above)
- Constant time, pure RT, elegant ✨

**For Rectangular Buildings** (L > W):
- Use **quadratic formula solution** (Option C)
- Single evaluation, no iteration
- More aligned with RT philosophy (stays in quadrance space)
- No derivatives needed (unlike Newton-Raphson)

**Implementation**:

```javascript
/**
 * Calculate hip roof height using pure algebraic solution (quadratic formula)
 * NO iteration, NO derivatives - pure rational trigonometry
 */
function calculateHipHeight_Algebraic(width, length, targetRoofArea) {
  const W = Math.min(width, length);   // Short dimension
  const L = Math.max(width, length);   // Long dimension
  const A = targetRoofArea;

  // Ridge length (deterministic from s=0.5 hip rafters)
  const ridgeLength = L - W;

  // Special case: Square building (ridgeLength ≈ 0)
  if (ridgeLength < 0.01) {
    // Pure pyramid - use direct solution from Case 1
    const slopeLength = A / (2 * W);
    const Q_slope = slopeLength * slopeLength;
    const Q_halfBase = (W / 2) * (W / 2);
    const Q_height = Q_slope - Q_halfBase;

    if (Q_height <= 0) {
      console.error("[WOMBAT] Pyramid impossible: roof area too small");
      return { height: 0, ridgeLength: 0, isValid: false };
    }

    return {
      height: Math.sqrt(Q_height),
      ridgeLength: 0,
      ridgeOffset: W / 2,
      span: W,
      achievedArea: A,
      isValid: true
    };
  }

  // Rectangular building - solve quadratic equation for u
  // where u = √(Q + W²/4) = main slope slant height

  // Quadratic coefficients: au² + bu + c = 0
  const a = 4 * L * (L - 2*W);
  const b = -4 * (L - W) * A;
  const c = A*A - W*W*W*W;  // A² - W⁴

  // Quadratic formula discriminant
  const discriminant = b*b - 4*a*c;

  if (discriminant < 0) {
    console.error("[WOMBAT] No real solution for hip roof height");
    return { height: 0, ridgeLength, isValid: false };
  }

  // Take positive root (negative root would be unphysical)
  const u = (-b + Math.sqrt(discriminant)) / (2 * a);

  if (u <= 0) {
    console.error("[WOMBAT] Invalid slant height (u ≤ 0)");
    return { height: 0, ridgeLength, isValid: false };
  }

  // Extract height quadrance from u
  const Q_height = u*u - (W*W/4);

  if (Q_height <= 0) {
    console.error("[WOMBAT] Invalid height quadrance (Q ≤ 0)");
    return { height: 0, ridgeLength, isValid: false };
  }

  const roofHeight = Math.sqrt(Q_height);

  // Verify area constraint (for validation)
  const slopeLength = u;  // u = √(Q + W²/4) = main slope slant height
  const hipRafterLength = Math.sqrt(u*u + W*W/4);  // Pythagorean relationship
  const achievedArea = 2 * ridgeLength * slopeLength + 2 * W * hipRafterLength;

  console.log(`[WOMBAT] Hip roof solved algebraically:`);
  console.log(`  Footprint: ${W.toFixed(2)}m × ${L.toFixed(2)}m`);
  console.log(`  Ridge length: ${ridgeLength.toFixed(2)}m`);
  console.log(`  Roof height: ${roofHeight.toFixed(2)}m`);
  console.log(`  Target area: ${A.toFixed(2)}m²`);
  console.log(`  Achieved area: ${achievedArea.toFixed(2)}m²`);
  console.log(`  Error: ${Math.abs(achievedArea - A).toFixed(4)}m²`);

  return {
    height: roofHeight,
    ridgeLength: ridgeLength,
    ridgeOffset: W / 2,
    span: W,
    achievedArea: achievedArea,
    isValid: Math.abs(achievedArea - A) < A * 0.01  // 1% tolerance
  };
}
```

**Why this is elegant**:
1. ✅ **No iteration** - single evaluation
2. ✅ **No derivatives** - pure algebra
3. ✅ **Works in Q-space** - stays rational as long as possible
4. ✅ **Single sqrt** - only at final extraction of height
5. ✅ **Geometric insight** - recognizes Pythagorean relationship between slope and hip rafter
6. ✅ **Wildberger-approved** - uses quadrance and spread concepts throughout

This is the "clever algebra" solution! 🎯

### Hip Roof Surface Area Components (Reference)

1. **Two main slopes** (trapezoidal):
   - Each trapezoid: base₁ = ridgeLength, base₂ = L, height = slopeLength
   - Slant height: `slopeLength = √(Q + W²/4)`
   - Area: `2 × ridgeLength × slopeLength`

2. **Four hip end triangles**:
   - Each triangle: base = W/2, slant height = hipRafterLength
   - Hip rafter: `hipRafterLength = √(Q + W²/2)`
   - Area: `4 × (W/2 × hipRafterLength / 2) = W × hipRafterLength`

**Total Area**:
```
A_hip = 2·ridgeLength·√(Q + W²/4) + 2W·√(Q + W²/2)
```

Where `ridgeLength = L - W` (deterministic from footprint).

## Prismatic Integration Challenge

**Core Issue**: Hip roofs are NOT prismatic extrusions.

Our current architecture:
```javascript
// Current pattern (works for gable/shed):
1. Solve 2D profile (cross-section perpendicular to ridge)
2. Extrude profile along ridge direction (long axis)

```

**Hip roofs break this pattern**:
- No single 2D profile can be extruded to create hip geometry
- Hip ends taper to points (ridge endpoints), not uniform extrusion

```javascript

  };
```

## Code Structure (Section19.js)

### New Functions Required

```javascript
/**
 * Calculate hip roof geometry from roof area constraint
 * Uses binary search to find height that satisfies area
 * @param {number} width - Building width (short dimension)
 * @param {number} length - Building length (long dimension)
 * @param {number} targetRoofArea - Target roof area from d_85
 * @returns {Object} - { height, ridgeLength, ridgeOffset, achievedArea, isValid }
 */
function calculateHipHeight(width, length, targetRoofArea) {
  const span = Math.min(width, length);
  const maxDimension = Math.max(width, length);

  // Ridge geometry (deterministic from 45° hip rafters)
  const hipRidgeLength = maxDimension - span;  // Shortened by span/2 at each end
  const ridgeOffset = span / 2;                 // Distance from end to ridge endpoint

  // Binary search on height
  let hMin = 0;
  let hMax = span * 2;  // Reasonable upper bound
  const tolerance = 0.01;  // 1cm precision
  const maxIterations = 50;

  let bestHeight = 0;
  let bestArea = 0;

  for (let i = 0; i < maxIterations; i++) {
    const h = (hMin + hMax) / 2;

    // Calculate roof area with this height
    const slopeLength = Math.sqrt(h*h + (span/2)*(span/2));
    const mainSlopeArea = 2 * hipRidgeLength * slopeLength;

    const hipRafterQuadrance = (span/2)*(span/2) + ridgeOffset*ridgeOffset + h*h;
    const hipRafterLength = Math.sqrt(hipRafterQuadrance);
    const hipEndArea = 2 * span * hipRafterLength;

    const calculatedArea = mainSlopeArea + hipEndArea;

    bestHeight = h;
    bestArea = calculatedArea;

    if (Math.abs(calculatedArea - targetRoofArea) < tolerance) {
      break;  // Converged
    } else if (calculatedArea < targetRoofArea) {
      hMin = h;  // Need taller roof
    } else {
      hMax = h;  // Need flatter roof
    }
  }

  return {
    height: bestHeight,
    ridgeLength: hipRidgeLength,
    ridgeOffset: ridgeOffset,
    span: span,
    achievedArea: bestArea,
    isValid: Math.abs(bestArea - targetRoofArea) < targetRoofArea * 0.05  // 5% tolerance
  };
}

/**
 * Generate 3D nodes for hip roof geometry
 * @param {Object} hipData - Result from calculateHipHeight()
 * @param {Object} footprint - { width, length }
 * @param {number} wallHeight - Height of walls
 * @returns {Object} - { ground, eave, ridge }
 */
function generateHipNodes3D(hipData, footprint, wallHeight) {
  const W = footprint.width;
  const L = footprint.length;
  const h = wallHeight;
  const ridgeHeight = wallHeight + hipData.height;

  const halfW = W / 2;
  const halfL = L / 2;

  // Ground nodes (Z=0)
  const ground = [
    { x: -halfW, y: -halfL, z: 0 },
    { x: +halfW, y: -halfL, z: 0 },
    { x: +halfW, y: +halfL, z: 0 },
    { x: -halfW, y: +halfL, z: 0 },
  ];

  // Eave nodes (Z=wallHeight)
  const eave = [
    { x: -halfW, y: -halfL, z: h },
    { x: +halfW, y: -halfL, z: h },
    { x: +halfW, y: +halfL, z: h },
    { x: -halfW, y: +halfL, z: h },
  ];

  // Ridge endpoints (Z=wallHeight+roofHeight)
  const ridge = [];
  if (hipData.ridgeLength > 0.01) {
    // Rectangular building - two ridge endpoints
    const halfRidge = hipData.ridgeLength / 2;
    ridge.push({ x: 0, y: -halfRidge, z: ridgeHeight });
    ridge.push({ x: 0, y: +halfRidge, z: ridgeHeight });
  } else {
    // Square building - single apex (pyramid)
    ridge.push({ x: 0, y: 0, z: ridgeHeight });
  }

  return { ground, eave, ridge };
}
```
